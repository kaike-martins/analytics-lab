# 📂 analytics-lab

Repositório de projetos práticos em análise de dados, desenvolvidos durante minha formação em Ciência de Dados na FMU São Paulo.

Cada projeto aborda um problema real ou simulado de negócio, aplicando Python, pandas, visualização de dados e raciocínio analítico para gerar insights acionáveis.

---

## 🗂️ Projetos

| Projeto | Descrição | Ferramentas |
|---|---|---|
| [analise-vendas-erva-mate](./analise-vendas-erva-mate/) | Análise exploratória de vendas B2B com foco em desempenho por vendedor, produto e estado | Python, Pandas, Matplotlib |

> Novos projetos serão adicionados conforme o avanço da formação.
